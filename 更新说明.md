```javascript
// 计算周数（以 3 月 1 日为基准，该周为第一周）
function getWeekNumber(date) {
    // 基准日期：2026 年 3 月 1 日（周日）
    const baseDate = new Date(2026, 2, 1); // 月份从 0 开始，2 表示 3 月
    const baseWeekStart = getWeekStart(baseDate);
    
    const targetWeekStart = getWeekStart(date);
    
    // 计算两个日期之间的毫秒差
    const diffTime = targetWeekStart.getTime() - baseWeekStart.getTime();
    // 转换为周数（向上取整，因为 3 月 1 日那周是第一周）
    const diffWeeks = Math.round(diffTime / (7 * 24 * 60 * 60 * 1000));
    
    return diffWeeks + 1; // +1 因为 3 月 1 日那周是第一周
}
```

## 功能说明

1. **周数计算基准**：以 2026 年 3 月 1 日结束的那一周为"第 1 周"
2. **显示格式**：日期范围 + 周数徽章，例如 "2/23 ~ 3/1 第 1 周"
3. **排序规则**：按周倒序排列，越新的周显示在越上面
4. **徽章样式**：浅棕色背景 (#8B7355)，米白色文字 (#FFFAE8)，圆角徽章
