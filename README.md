# 纸黄待办 - 项目文件

## 项目简介
- **名称**：纸黄待办
- **类型**：待办事项 + 记账网页应用
- **技术**：纯 HTML + CSS + JavaScript（零成本运行）
- **托管**：GitHub Pages

## 文件说明
- `index.html` - 主程序文件（直接在浏览器打开即可使用）
- `project.md` - 项目文档

## 本地使用
直接用浏览器打开 `index.html` 即可

## 在线访问
https://cynic-joe.github.io/todo-web

## 功能列表
- 待办事项（添加/删除/完成）
- 待办显示添加时间
- 已完成列表（按周分组）
- 记账功能（收入/支出）
- GitHub Gist 云同步
- 自动同步/自动拉取
- 响应式设计（手机/电脑）

## 部署到 GitHub Pages
1. 创建 GitHub 仓库
2. 上传 index.html
3. Settings → Pages → Deploy from main branch
